# Funparks Website

The official marketing website for the Funparks app.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import repo
3. Click Deploy — done!

## Update App Store URL

In `components/DownloadSection.js`, replace the IOS constant with your real App Store ID when published.
